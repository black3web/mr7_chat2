# MR7 Chat - Changelog

## [1.0.0] - 2025-04-18

### Initial Release

#### Features
- Real-time private messaging (Firebase Firestore)
- Group chats with full admin permissions system
- Stories (48-hour, max 3 per cycle, photos + videos)
- 7 AI services: Gemini 2.5 Flash, DeepSeek (V3.2/R1/Coder),
  Nano Banana 2, NanoBanana Pro, Seedance AI, Kilwa Video, AI Music
- Full Arabic/English localization with real-time switching
- Dark red/black theme with animated geometric background
- Glass morphism UI throughout
- Developer admin panel with statistics, user management,
  broadcast system, AI service toggle controls
- Message reactions, edit, delete, reply, forward
- Story reactions + comments (sent as private messages)
- Multi-account support with instant switching
- Profile customization (photo, username, bio, password)
- Privacy settings (hide last seen, online status, name, photo)
- Support chat with developer (image + text)
- Search users and groups by username or name
- GitHub Actions CI/CD for APK + GitHub Pages web deploy
- Firebase Firestore security rules
- Firebase Storage security rules
- ProGuard rules for release APK

#### AI Services
- **Gemini 2.5 Flash** - Real-time AI chat
- **DeepSeek** - Multi-model chat with conversation memory
- **Nano Banana 2** - 2K image generation
- **NanoBanana Pro** - 1K/2K/4K image generation and editing
- **Seedance AI** - Text/Image to Video (1.0 Lite/Pro, 1.5 Pro)
- **Kilwa Video** - Fast text-to-video (Seedance 1.5 Pro)
- **AI Music** - Text-to-music with mood tags (Visco AI)
